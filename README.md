# RodasHackathonCCR
Projeto Web, desenvolvido em js, e Mobile, em python, para o desafio Hackathon CCR com o objetivo de melhorar a qualidade de vida de Caminhoneiros e Caminhoneiras. Desafio Proposto pela CCR e Shawee
